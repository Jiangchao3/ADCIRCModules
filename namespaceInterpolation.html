enum Method
Enumerator
NoMethod 

No interpolation.

Average 

Mesh sized based averaging defined by the mesh size and any multipliers.

Nearest 

Nearest node interpolation within specified radius defined by the mesh size and any multipliers.

Highest 

Greatest value within specified radius.

PlusTwoSigma 

Calculates the average of values > 2 sigma from the mean. Used to interpolate raised features withinout overstating.

BilskieEtAll 

Uses the Bilskie et al radius sizing. Generally 1/4 the average element size or the nearest point for coarse regions.

InverseDistanceWeighted 

Uses an inverse distance weighting approach for the points available within the radius defined by the mesh size and any multipliers.

InverseDistanceWeightedNPoints 

Uses an inverse distance weighting approach but does not require a specified radius size. Instead, the nearest 'filterSize' (i.e. number) of points will be selected.

AverageNearestNPoints 

Creates a selection box that will select and average the nearest 'filterSize' (i.e. number of) points.
