Detailed Description

Author
Zach Cobell

The QADCModules_errors class is designed to be used by all other classes in this suite of classes to relay error messages. Error messages are mapped to a set of descriptions on startup so that the user can easily understand where a particular function has failed. All routines must return an error code as defined by this class or no error (ERROR_NOERROR) to achieve successful completion
