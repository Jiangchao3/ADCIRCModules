Detailed Description

Class that serves as an interface to the standard C PROJ4 library.

This function serves as the C++ interface to the C library PROJ4 (which is compiled into this code)

Projection ( )
explicit

Constructor for the proj4 wrapper class.

Constructs an object used to convert coordinates. The initialization function is run at startup which parses the included EPSG file (:/rsc/epsg) to determine the parameters to pass to the Proj4 API
